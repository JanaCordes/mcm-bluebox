BB-Insights - BlueBox Analytics

Step 1: Open the Node-RED editor and create your query

BB-Insights comes with a Node-RED instance that is connected to your metadata warehouse. You can create SQL queries, pull in external data sources, and use javascript functions to perform any possible analysis on the data.

After you created your flow in Node-RED, turn it into a data source for BB-Insights. Use the HTTP node to create an endpoint and give it any name you like. Connect your flow to the HTTP node and then come back here.

Open the Node-RED editor
Step 2: Enter the endpoint name for your Node-RED data source

Enter the name you specified for your HTTP node in Node-RED. You only need to enter the name of the endpoint, BB-Insights already knows the rest of the URL.

{{nodered.url}}/
Step 3: Select a suitable visualization for your data

BB-insights uses Bokeh and D3.js to visualize your data. You can now choose which type of visualization you want to use.

Each type supports different data structures. All data from Node-RED has to come in tables or matrixes, the different visualization types support different amounts of rows and columns.
